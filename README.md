
# Moldi's Pizza UI

Welcome to Moldi's Pizza (UI only). This React application constitutes the user interface for the "Moldi's Pizza" repository.

- Sign in
- Pizzas
- Forgot password
- Not found
- Profile
- Orders
- Custom pizza page with reviews
- Checkout
- Payment success
